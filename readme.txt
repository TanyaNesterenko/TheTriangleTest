Проект тестирования веб-формы "Опеределение вида треугольника"
(https://team.cft.ru/triangle/zadanie/triangle.html?token=5011cd2bc80f4c58a36bbf39a573e481).

Запуск всех тестов происходит в классе triangle.suits.Suits.
В пакете triangle.suits располагаются:
1) общий тест-сет;
2) тест-сет для прогона позитивных тестов на проверку правильности вводимых данных
3) тест-сет для прогона негативных тестов на проверку правильности вводимых данных
4) тест-сет на проверку результатов
5) тест-сеты на проверку результатов по типам треугольника

По отдельности тесты можно запускать в пакете cases, непосредственно в классе теста.
Логи прогона тестов находятся в файле log.txt.
Для запусков тестов необходим Google Chrome и установленный веб-драйвер chromedriver.

В качестве инструментов для автоматизации был выбран Selenium WebDriver + JUnit.
Selenium наиболее удобный, простой и современный инструмент для тестирования web-страниц.
